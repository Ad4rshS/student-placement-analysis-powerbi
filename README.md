# Student Placement Analysis – Power BI

## Project Overview
This project analyzes student placement data to identify trends in salary distribution, department-wise performance, company-wise hiring, and gender-based insights using Power BI.

## Key Objectives
- Analyze average salary across departments and companies
- Understand placement distribution by department and company
- Compare placement and salary patterns by gender

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard Pages
1. Salary Analysis by Department and Company  
2. Department & Company-Wise Placement Distribution  
3. Gender-Wise Placement and Salary Analysis  

## Key Insights
- Average salary varies significantly across departments.
- Placements are concentrated among a limited number of companies.
- Gender-wise differences exist in placement participation and salary.

## Files Included
- `Placement_Analysis.pbix` – Power BI dashboard file
- `screenshots/` – Dashboard preview images
- `data/` – Dataset used for analysis 

## Notes
This project is created for learning and portfolio purposes using academic (demo)placement data.
